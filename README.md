1. Создан класс BankAccount:
Поля класса:
· имя владельца (строка);
· баланс (целочисленный);
· дата открытия (LocalDateTime);
· поле-флажок заблокирован ли счет (boolean).

2. Добавлен один конструктор, который принимает только один параметр - имя владельца.
Остальные поля инициализированы начальными значениями.

3. Добавлены методы:
· deposit (amount) – пополнение счета;
· withdraw (amount) – снятие денег;
· transfer (otherAccount, amount) – перевод денег на другой счет.
Каждый метод возвращает значение типа boolean - успешно или неуспешно была проведена операция.

4. Проверены реализованные методы

Доп.задания:
1. В класс добавлено поле number - номер счета в формате XXXXXXXX, где X - цифра от 0 до 9.
Во время создания объекта оно генерируется рандомно.
2. Реализован метод equals для сравнения счетов.
